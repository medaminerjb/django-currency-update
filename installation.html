Installation¶

  1. Install the library:

    pip install django-currency-updater

  2. Add django_currency_updater to your INSTALLED_APPS in settings.py:

    INSTALLED_APPS = [
...
'django_currency_updater',
]

  3. Run migrations:

    python manage.py migrate
